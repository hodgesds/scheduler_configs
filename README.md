# scheduler_configs
This repo contains various scheduler configs for
[`sched_ext`](https://github.com/sched-ext/sched_ext). The `scx_layered`
directory contains scheduler configs for
[`scx_layered`](https://github.com/sched-ext/scx/tree/main/scheds/rust/scx_layered).
For various applications. Note, these are experimental scheduler configs and
may cause performance regressions, so use them at your own risk.
